О приложении


На главную


Разрабатываемый web-интерфейс предназначен для работы с базой данных «Преподаватели и сотрудники ДПК». Он обеспечивает отображение таблиц базы данных «Преподаватели и сотрудники ДПК», редактирование главной таблицы, которая содержит все данные о преподавателях и сотрудниках, добавление новых записей в таблицу, удаления выбранных строк таблицы, а также быстрый поиск по главной таблице. Так же есть вохможность редактировать и другие таблицы базы данных.

Основные преимущества MySQL:

  • Многопоточность, поддержка нескольких одновременных запросов;
  • оптимизация связей с присоединением многих данных за один проход;
  • записи фиксированной и переменной длины;
  • ODBC драйвер;
  • гибкая система привилегий и паролей;
  • гибкая поддержка форматов чисел, строк переменной длины и меток времени;
  • интерфейс с языками C и Perl, PHP;
  • быстрая работа, масштабируемость;
  • совместимость с ANSI SQL;
  • бесплатна в большинстве случаев;
  • хорошая поддержка со стороны провайдеров услуг хостинга;
  • быстрая поддержка транзакций через механизм InnoDB.
